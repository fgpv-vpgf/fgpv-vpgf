Class Map

Provides controls for modifying the map, watching for changes, and to access map layers and UI properties.

example

Acting on a double click event

 
mapInstance.doubleClick.subscribe(mouseEvent => {
 console.log(`Double click at pixel location (${mouseEvent.pageX}, ${mouseEvent.pageY})`);
});

setExtent

  • setExtent(extent: Extent, fit?: boolean): void
    • Defined in api/src/map.ts:136

    Sets the extent of the map. The extent must be in the same spatial reference as the map. Note: the extent will not be modified.

    The extent property must be provided using standard ESRI JSON format. The fit property is optional (defaults to false). If true, for maps containing tile layers the input extent always shown completely on the map.

    Parameters

    • extent: Extent
    • Default value fit: boolean = false

    Returns void
