constructor

  • new Polygon(id: string | number, elements: Array<Point | XY | XYLiteral | Array<Point | XY | XYLiteral>>, styleOptions?: PolygonStyleOptions): Polygon

  • Overrides BaseGeometry.constructor

    • Defined in api/src/geometry.ts:542

    Constructs a Polygon from the given array of points or ring coordinates.

    The different style options and values available are the following:

    • outlineColor: string hex code. default is '#AFAFAF'.
    • outlineWidth: number in pixels. default is 3.
    • fillColor: string hex code. default is '#000000'.
    • fillOpacity: number between 0 and 1. default is 0.

    NOTE: if linear rings are added to a polygon after it has been added to the map, it will not redraw TODO: add option for polygon icon fill.

    Parameters

    • id: string | number
    • elements: Array<Point | XY | XYLiteral | Array<Point | XY | XYLiteral>>
    • Optional styleOptions: PolygonStyleOptions
